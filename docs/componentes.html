Componentes

Componentes são os blocos de construção reutilizáveis do nosso Design System. Cada componente é cuidadosamente projetado, testado e documentado para garantir consistência, acessibilidade e facilidade de uso em todos os nossos produtos.

O que são componentes?

Componentes são elementos de interface independentes e reutilizáveis que encapsulam funcionalidade, comportamento e estilo. Eles podem ser tão simples quanto um botão ou tão complexos quanto um modal com múltiplas interações.

Cada componente em nosso sistema:

  • Segue nossos princípios de design e diretrizes de marca
  • É totalmente acessível e testado com tecnologias assistivas
  • Possui variações para diferentes contextos e necessidades
  • Inclui documentação completa com exemplos de código
  • É responsivo e funciona em diferentes tamanhos de tela

Categorias de componentes

Organizamos nossos componentes em categorias para facilitar a navegação:

Básicos

Elementos fundamentais que servem como base para construções mais complexas.

  • Button - Ações primárias, secundárias e terciárias
  • Input - Campos de texto, números, senhas
  • Checkbox - Seleções múltiplas
  • Radio - Seleções únicas
  • Toggle - Alternadores on/off
  • Link - Navegação e referências

Formulários

Componentes especializados para coleta e validação de dados.

  • Select - Menus dropdown para seleção
  • Textarea - Campos de texto multilinha
  • Date Picker - Seleção de datas
  • File Upload - Envio de arquivos
  • Search - Campos de busca com autocompletar
  • Form - Container com validação integrada

Navegação

Elementos que ajudam usuários a se moverem pela aplicação.

  • Navigation Bar - Menu principal de navegação
  • Breadcrumb - Trilha de navegação hierárquica
  • Tabs - Organização de conteúdo em abas
  • Pagination - Navegação entre páginas de conteúdo
  • Sidebar - Menu lateral para navegação secundária
  • Steps - Indicadores de progresso em fluxos

Feedback

Componentes que comunicam status, confirmações e alertas.

  • Alert - Notificações e avisos contextuais
  • Toast - Mensagens temporárias de feedback
  • Badge - Indicadores de status ou contagem
  • Progress Bar - Visualização de progresso
  • Skeleton - Estados de carregamento
  • Spinner - Indicadores de processamento

Exibição de dados

Elementos para apresentar informações de forma estruturada.

  • Table - Exibição tabular de dados
  • Card - Containers para agrupar informações
  • List - Listas ordenadas e não ordenadas
  • Accordion - Conteúdo expansível
  • Stat - Métricas e estatísticas destacadas
  • Avatar - Representações de usuários

Sobreposições

Componentes que aparecem sobre o conteúdo principal.

  • Modal - Diálogos que requerem atenção
  • Drawer - Painéis laterais deslizantes
  • Popover - Conteúdo contextual flutuante
  • Tooltip - Dicas e informações adicionais
  • Dropdown Menu - Menus de ações contextuais
